# Git ハンズオン研修

新入社員向け Git / GitHub ハンズオン研修のプレビュー & 進捗管理アプリです。

## 概要

ハンズオン研修で使用するマークダウン資料を、ブラウザ上で見やすく表示します。
講師がリアルタイムで受講者の進捗を把握できるダッシュボード機能を備えています。

### 研修内容

- **Day 0** — Git for Windows のダウンロードとインストール
- **Day 1** — Git基本操作（GitHubアカウント作成、add / commit / push）
- **Day 2** — ブランチとPull Request

### 主な機能

- マークダウンプレビュー（コラムの折りたたみ表示、HTML埋め込み対応）
- 講師によるハンズオン選択（トップページで Shift+Click）
- 選択されたハンズオンのみ受講者がアクセス可能
- 受講者の座席番号による識別（ヘッダーに表示、クリックで座席クリア）
- セクションごとの「できた」チェックポイント
- コマンド実行ごとのOK / エラー報告（通し番号付き）
- 「助けてほしい」ヘルプシグナルボタン
- 講師ダッシュボード（リアルタイム進捗マトリクス、着席状態、ヘルプリクエスト通知）

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:3000 でアプリが起動します。

## 使い方

### 講師

1. `http://localhost:3000` のトップページを開く
2. 開催するハンズオン（Day 0〜2）を **Shift+Click** で選択（もう一度 Shift+Click で解除）
3. 選択中のパネルは青くハイライトされ「開催中」バッジが表示される
4. `http://localhost:3000/dashboard` で全受講者の進捗・着席状態をリアルタイム確認
5. ハンズオンに講師として参加する場合は、座席選択画面で「講師として参加」をクリック

### 受講者

1. `http://<講師のIP>:3000` にアクセス
2. 講師が選択したハンズオンをクリック（未選択のハンズオンには参加不可）
3. 座席番号（1〜15）を入力して「開始する」をクリック
4. ハンズオンを進めながら、コマンド実行後に「OK」または「エラーが出た」を報告
5. 困ったら右下の「助けてほしい」ボタンを押す
6. ヘッダーの座席バッジをクリックすると、座席をクリアしてトップに戻れる

### テスト

同じブラウザで複数の座席をテストする場合は、URLパラメータを使います。

```
http://localhost:3000/day1?seat=0   # 講師
http://localhost:3000/day1?seat=1   # 座席1
http://localhost:3000/day1?seat=2   # 座席2
```
